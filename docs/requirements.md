# GraphSpeak 需求说明文档

## 项目概述

GraphSpeak 是一个 AI 驱动的绘图工具，用户通过自然语言对话生成流程图等图表，支持多种图形类型和手动编辑。单用户使用，无需登录注册。

---

## 核心功能

### 1. 图形绘制

| 功能 | 描述 |
|------|------|
| AI 生成 | 通过对话生成各类图形 |
| 手动编辑 | 支持拖拽、修改、删除图形元素 |
| 支持类型 | 流程图、架构图、时序图、类图、ER图等 |
| 样式控制 | 根据用户描述 AI 自动决定样式（颜色、形状等） |

### 2. 画布操作

| 功能 | 描述 |
|------|------|
| 无限画布 | 无尺寸限制，默认范围可扩大 |
| 缩放 | 滚轮放大/缩小 |
| 平移 | 拖拽画布移动 |

### 3. 项目管理

| 功能 | 描述 |
|------|------|
| 多项目 | 支持创建多个独立项目（一个项目 = 一个画布） |
| 新建项目 | 从空白画布开始 |
| 复制项目 | 复制现有项目 |
| 删除项目 | 删除指定项目 |
| 重命名项目 | 修改项目名称 |
| 项目排序 | 按修改时间倒序 |
| 自动保存 | 画布修改后自动保存或手动保存（Ctrl+S） |

### 4. 文件操作

| 功能 | 描述 |
|------|------|
| 保存 | 保存当前项目到数据库（支持自动保存和手动保存） |
| 导出 | 前端导出为 PNG 图片 |
| 导入 | 导入 mxGraph XML 文件 |
| 格式支持 | mxGraph XML + JSON |

### 5. AI 对话

| 功能 | 描述 |
|------|------|
| 自然语言交互 | 通过文字描述生成/修改图形 |
| 上下文记忆 | 保留当前会话的完整上下文 |
| 错误处理 | AI 返回错误提示并询问 |
| 对话记录 | 保存到数据库 |
| 会话管理 | 支持多个会话列表 |
| 删除会话 | 删除指定会话 |
| 会话命名 | AI 基于第一条消息异步生成会话名称 |

## 界面布局（Draw.io 风格）

```
┌────────────────────────────────────────────┐
│ [项目▼]  [保存]  [导出]                    │
├────────────────────────────────────────────┤
│                                            │
│                                            │
│            绘图区域                        │
│            (最大空间)                      │
│                                            │
│                                            │
│                                            │
├────────────────────────────────────────────┤
│ [会话▼]  [💬] "画一个用户注册流程图..."      │
└────────────────────────────────────────────┘
```

**交互方式**：
- 项目列表：左上角"项目"按钮，点击展开悬浮列表
- 会话列表：底部"会话"按钮，点击展开悬浮列表
- AI对话：底部固定栏，可展开/折叠

---

## 交互流程

### AI 生成图形流程

```
用户输入描述 → 后端处理 → AI 解析 → 生成结构化数据 → 前端渲染 → 用户可手动编辑
```

### AI 修改图形流程

```
用户输入修改指令 → AI 读取当前画布 → 解析指令 → 返回修改操作 → 前端执行修改
```

|--------|----------|
| 项目数据 | MySQL（图形 XML/JSON） |
| 聊天记录 | MySQL |
| 会话信息 | MySQL |

## 非功能需求

| 需求 | 描述 |
|------|------|
| 单用户 | 无需用户系统 |
| 响应速度 | AI 生成图形响应时间 < 10 秒 |
| 数据持久化 | 所有操作保存到数据库 |
